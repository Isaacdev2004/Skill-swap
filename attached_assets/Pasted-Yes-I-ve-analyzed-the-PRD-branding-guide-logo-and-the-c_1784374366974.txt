# REPLIT AI MASTER PROMPT (PHASE 1)

## Project Overview

You are a senior React engineer and UI/UX architect.

I want you to build a production-quality frontend for a SaaS web application called **SkillSwap**.

This is **NOT** a prototype.

Everything should be production-ready, scalable, reusable, responsive and clean.

The frontend should look like something built by a senior designer from Stripe, Linear, Notion or Vercel.

Avoid generic AI-looking interfaces.

The application is a peer-to-peer skill exchange platform where users exchange knowledge instead of paying money.

Example:

User A teaches Python and wants Guitar.

User B teaches Guitar and wants Python.

The platform matches them and allows them to schedule sessions and chat.

The actual lesson happens through a Google Meet link pasted into chat.

There is NO built-in video calling.
